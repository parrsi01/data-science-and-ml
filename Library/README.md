# Data Science Reference Library

> A structured, research-level reference of every concept,
> definition, and command used in this repository.
> Written for beginners. Formatted for mobile reading.

---

## How to Use This Library

Each file covers one topic area. Definitions follow
a consistent structure: what it is, why it matters,
and a concrete example. Open any file and scroll
to find the term you need.
