# Decentralized Federated Anomaly Detection using MARL + XGBoost

Institutional module for Simon Parris' decentralized federated anomaly detection algorithm with MARL-driven peer communication and XGBoost local models.

- Code: `algorithm_marl_xgboost/src/`
- Docs: `algorithm_marl_xgboost/docs/`
- Config: `algorithm_marl_xgboost/configs/experiment.yaml`
- Reports: `algorithm_marl_xgboost/reports/`
- Logs: `algorithm_marl_xgboost/logs/experiment.jsonl`

Run:

```bash
venv/bin/python -m algorithm_marl_xgboost.src.run_experiment --config algorithm_marl_xgboost/configs/experiment.yaml
```
